Motor de ejercicios parametrizados
Generacion dinamica + solucion dual docente-estudiante + pistas progresivas + adaptacion de dificultad
EJP-01El sistema genera ejercicios automaticamente con valores aleatorios dentro de rangos definidos (ej: a en [1,10], b en [2,20]).
EJP-02El profesor define la plantilla del ejercicio con parametros variables (ej: 'Calcular el MCD de {a} y {b}').
EJP-03El profesor establece rangos, restricciones y condiciones de cada parametro (ej: a debe ser primo, b divisible por 3).
EJP-04El sistema adapta la dificultad de los valores generados segun el nivel del estudiante.
EJP-05El profesor carga el paso a paso oficial de solucion con formulas LaTeX para cada plantilla de ejercicio.
EJP-06El estudiante escribe su propio paso a paso de resolucion en un editor matematico con soporte LaTeX.
EJP-07El sistema muestra pistas progresivas configuradas por el docente antes de revelar la solucion oficial.
EJP-08El estudiante puede elegir ver la solucion oficial paso a paso en cualquier momento.
EJP-09Soportar ejercicios parametrizados para todos los temas: aritmetica, algebra, geometria, trigonometria y razonamiento matematico.
EJP-10El sistema verifica automaticamente la respuesta final del estudiante y la marca como correcta o incorrecta.
EJP-11El sistema genera una variante nueva del mismo ejercicio cada vez que el estudiante lo repite.
EJP-12El profesor puede previsualizar las variantes generadas antes de publicar el ejercicio.
EJP-13El sistema registra el historial de intentos del estudiante por cada plantilla de ejercicio.
EJP-14El sistema permite usar ejercicios parametrizados dentro de examenes y simulacros (cada estudiante recibe valores unicos).
EJP-15El sistema soporta formulas LaTeX en la plantilla, en el paso a paso del docente y en el paso a paso del estudiante.
EJP-16El sistema genera flashcards/tarjetas de repaso automaticas a partir de los ejercicios parametrizados.
