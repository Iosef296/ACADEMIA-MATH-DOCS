# Comportamiento - Diagramas de Comportamiento 
 
Carpeta dedicada a los diagramas de comportamiento del sistema. 
Documenta como interactuan los actores con la plataforma. 
 
## Subcarpetas 
 
- `1-Casos_Uso/`: 18 Diagramas de Casos de Uso (CU01-CU18) 
- `2-Secuencias/`: 18 Diagramas de Secuencia (DS01-DS18) 
- `3-Estados/`: 18 Diagramas de Estado (DE01-DE18) 
- `Especificaciones/`: 18 Especificaciones tecnicas (ECU01-ECU18) 
 
## Relacion entre diagramas 
 
- **Casos de Uso**: Que puede hacer el usuario 
- **Secuencias**: Como se ejecuta paso a paso 
- **Estados**: Que estados atraviesa un objeto
