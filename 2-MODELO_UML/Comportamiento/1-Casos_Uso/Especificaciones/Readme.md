# Especificaciones - Especificaciones de Casos de Uso 
 
18 especificaciones tecnicas de casos de uso del sistema Academia Virtual de Matematicas. 
 
## Que son? 
 
Cada especificacion detalla tecnicamente un caso de uso: 
- Precondiciones: Que debe cumplirse antes de ejecutar 
- Flujo principal: Paso a paso de la ejecucion normal 
- Flujos alternativos: Escenarios de error o excepciones 
- Postcondiciones: Estado del sistema despues de ejecutar 
- Reglas de negocio: Validaciones y restricciones 
- Requisitos no funcionales: Rendimiento, seguridad, usabilidad
